# Library Management System (WebAssembly)

## Description
This project is a mini Library Management System developed using C++. It is converted into a web-based application using WebAssembly with the help of Emscripten. The main aim of this project is to apply OOP concepts, file handling, and basic data management in a practical way.

## Features
- Add new books
- Issue books to students
- Return books
- Search books using ID or title
- Display issued books
- Show count of available and issued books
- Update and delete book details
- Store data using file handling

## Concepts Used
- Classes and Objects
- Encapsulation
- STL (vector, string)
- File handling using fstream
- Menu-driven program

## Technologies Used
- C++
- Emscripten (for WebAssembly)
- HTML (for browser interface)

## How to Run
1. Open the website link
2. Follow the menu displayed on the screen
3. Enter inputs step by step when prompted

## Project Files
- main.cpp → program logic
- index.html → browser page
- index.js → generated by Emscripten
- index.wasm → compiled file

## Note
This project runs inside the browser using WebAssembly. Some data may not persist permanently depending on the environment.
